# Detectives de la probabilidad: ¿puede el azar engañarnos?

Repositorio con la documentación del caso integrador de la asignatura Complementos para la Formación Disciplinar de las Matemáticas, del Máster Universitario en Formación del Profesorado.

## Descripción

La actividad está diseñada para 1.º de Bachillerato y trabaja contenidos de estadística, probabilidad y probabilidad condicionada mediante una metodología activa basada en gamificación.
